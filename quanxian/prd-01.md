**总体功能**：面向后台管理系统精细化权限管控需求，开发多级树形权限系统，实现部门、角色、菜单、按钮四级联动权限配置，精准管控用户操作权限，适配各类后台系统权限管理场景。

**详细功能需求**：以树形结构可视化展示部门、菜单、权限层级关系，支持节点拖拽调整层级与排序。实现节点勾选联动授权、批量权限配置，可根据用户角色动态渲染侧边菜单，支持按钮级精细化权限控制，可实时预览权限配置效果，精准区分不同角色的系统操作权限。

**技术要求**：Go Gin实现树形数据组装、多级权限校验、角色菜单绑定接口；Vue3实现树形拖拽、联动勾选、动态菜单渲染等高难度交互；SQLite存储权限、角色、菜单、部门数据；后端代码存放于 server 目录，前端代码存放于 web 目录，项目可直接运行。
