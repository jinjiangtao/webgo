针对平台文本检索优化需求，开发智能关键词分词检索系统，实现搜索联想、内容高亮、多条件筛选功能，大幅提升前端搜索交互体验与检索精准度。

**详细功能需求**：后台支持关键词库批量录入、分类管理与失效编辑维护。前端输入关键词即可实时触发联想下拉提示，快速预览匹配结果。系统自动对检索内容分词匹配，关键字高亮展示，支持多条件联动筛选、结果排序与分页动态加载，自动留存用户检索日志，持续优化搜索效果。

**技术要求**：Go Gin实现文本分词匹配、关键词检索、联想数据返回接口；Vue3实现实时联想提示、内容高亮渲染、联动筛选等高难度前端交互；SQLite存储关键词库与检索日志数据；后端代码存放于 server 目录，前端代码存放于 web 目录，项目可直接编译运行。
