适配线上评选、投票活动运营场景，开发多功能在线投票系统，支持活动自定义配置、用户在线投票、数据实时统计，搭载防刷票机制，保障投票活动公平有序开展。

**详细功能需求**：后台可自主创建投票活动，自定义投票选项、活动时长、参与权限。前端支持单选、多选投票模式，实时刷新票数数据与排名榜单。系统内置防重复投票机制，拦截恶意刷票、重复提交行为，通过图表可视化展示投票占比数据，支持活动启停管控、投票记录查询与数据汇总。

**技术要求**：Go Gin实现投票规则配置、防重校验、票数统计、活动管控接口；Vue3实现数据实时刷新、榜单动态排序、图表联动展示等高交互；SQLite存储活动信息与投票记录数据；后端代码存放于 server 目录，前端代码存放于 web 目录，项目可直接运行。
