面向小型设备运维监控场景，搭建设备状态可视化仪表盘，实现设备运行状态实时监测、异常告警、数据筛选与日志记录，满足轻量化设备运维管控需求。

**详细功能需求**：后台可录入设备基础信息、运行参数、设备类型。前端采用自适应卡片布局展示全部设备，在线、离线、异常三种状态自动变色标记。设备出现异常时触发弹窗告警提示，支持按设备类型、运行状态多条件筛选设备，实时刷新设备运行数据，自动留存设备运行日志，方便运维人员排查设备问题。

**技术要求**：Go Gin实现设备数据更新、状态检测、异常告警、日志记录接口；Vue3实现卡片动态渲染、状态联动变色、实时告警等高难度交互；SQLite存储设备信息与运行日志数据；后端代码存放于 server 目录，前端代码存放于 web 目录，项目可直接编译运行。
